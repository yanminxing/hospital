## 2 项目搭建

## 2.1 初始化项目

1 使用vite构建项目

1）使用命令

[创建说明文档](https://cn.vitejs.dev/guide/)

```npm
yarn create vite
```

2）选择vue和ts

3）下载依赖、运行项目

```
yarn
yarn dev
```
